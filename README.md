# Portfolio

Personal portfolio site for [jacobmaynard.dev](https://jacobmaynard.dev).

Built with TanStack Start, and deployed on Cloudflare Workers.

## Stack

- **Framework:** TanStack Start (React 19)
- **Styling:** Tailwind CSS v4
- **Animations:** GSAP, Motion, Lenis smooth scroll
- **Blog/Digest:** Markdown with vite-plugin-markdown
- **Hosting:** Cloudflare Workers
- **portless** For nice URLs during development
 
## Development

```bash
pnpm install
pnpm dev
```

## Deploy

```bash
pnpm run deploy
```
